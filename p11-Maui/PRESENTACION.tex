\begin{frame}
  \frametitle{¿Qué es?}
Maui es un planificador de tareas para clusters.\\
Se desarrolló a mediados de los 90.\\

Las políticas de planificación de Maui abarcan:
\begin{itemize}
\item Prioridad dinámica.
\item Reservas.
\item fairShare \emph{\footnotesize(Compartición justa)}
\end{itemize}
\end{frame}

\begin{frame}
  \frametitle{Funcionalidades}
  \begin{enumerate}
  \item \textbf{Control de Tráfico:} Gestiona las tareas para que no interfieran. 
  \item \textbf{Centrado en Objetivos:} Configuración de cluster según la tarea
  \item \textbf{Optimizaciones:} Supeditado a los dos puntos
    anteriores, el planificador intenta aprovechar los recursos del cluster.
  \end{enumerate}
\end{frame}

\begin{frame}
  \frametitle{Tipos de Nodos}
  SHARED
	Tasks from any combination of jobs may utilize available resources
SINGLEUSER
	Tasks from any jobs owned by the same user may utilize available resources
SINGLEJOB
	Tasks from a single job may utilize available resources
SINGLETASK
	A single task from a single job may run on the node
\end{frame}
